# :construction_worker: How to run
```bash
# Clone Repository
$ git clone https://github.com/neresleonardo/Proffy.git
```
### üì¶ Run API

```bash
# Go to server folder
$ cd Proffy/server

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```
Access API at http://localhost:3333/

### üíª Run Web Project

```bash
# Go to web folder
$ cd Proffy/web

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```
Go to http://localhost:3000/ to see the result.
